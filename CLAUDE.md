# プロジェクト固有の指示

## ユーザーへの選択肢提示

ユーザーに複数の選択肢から選んでもらう場合は、本文に番号付きリストを書くのではなく **`AskUserQuestion` ツールを使う**こと。

- 対象: 「どれから進めますか？」「A か B どちらにしますか？」のように、ユーザーの判断で分岐する場面
- 例外: 単一の yes/no 確認や、選択肢を提示しない通常の質問は本文で聞いてよい

## コーディング規約

### `let` 利用禁止

TypeScript コードでは **`let` を使わない**。常に `const` で宣言する。

- ループカウンタ等で再代入が必要な場面は `for...of` / `map` / `reduce` などで書き換える
- try/catch の結果を外に出したい時は、try ブロックをヘルパ関数に切り出して `return` する
- シングルトンの lazy init は `Map` や `WeakMap` でキャッシュする
